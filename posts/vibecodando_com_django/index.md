# Então... 

Decidi criar um produto do zero, sem validação de mercado e usando uma stack não tão "na moda" para entender os impactos da AI para criar produtos em nincho de mercado.

Irei registrar tudo e mais um pouco aqui como forma de entender e aprender em público qual o status quo do desenvolvimento de software até a presente data com o intuito de responder a seguinte questão:

> O Quão produtivo é a AI (LLM) para prototipar usando framework fullstack

Isso parte do principio que temos muitos testes e produtos criados em tecnologias que não são fullstacks como o Next.JS, mas pouquissimos testes documentados de como podemos obter melhores resultados ou piores se usarmos uma solução limitada, porém robusta que funcione muito bem numa VPS, afinal não queremos saber o quão é produtivo incluir AI no seu fluxo, queremos saber como usamos ela com fundamentos para ser a peça principal de desenvolvimentto, mas ainda orientada pelas mãos daquele que vos fala.
